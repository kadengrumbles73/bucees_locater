# Overview

I am building this to master data mapping and front-end interaction. This project helps me learn how to visualize complex datasets on interactive web maps.

This software uses Leaflet to map Buc-ee's locations. Click filter buttons to display stores with EV charging or car washes using a curated location dataset.
